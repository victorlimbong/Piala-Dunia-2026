# Template Input Prediksi Pertandingan

Gunakan template ini untuk meminta prediksi pertandingan sebelum pertandingan dimulai.

```markdown
Prediksi pertandingan:

- Kompetisi:
- Tahun:
- Fase:
- Tim A:
- Tim B:
- Tanggal:
- Lokasi:
- Zona waktu analisis:
- Mode analisis: Pre-Match
- Catatan tambahan:
```

---

## Contoh Pengisian

```markdown
Prediksi pertandingan:

- Kompetisi: Piala Dunia
- Tahun: 2026
- Fase: Round of 16
- Tim A: Brasil
- Tim B: Pantai Gading / Norwegia
- Tanggal: 5 Juli 2026
- Lokasi: [isi stadion dan kota]
- Zona waktu analisis: Asia/Jakarta
- Mode analisis: Pre-Match
- Catatan tambahan: prioritaskan form turnamen, ranking, Elo, cedera, line-up, dan analisis taktis
```

---

## Checklist Data yang Harus Dicari AI

- [ ] Jadwal resmi
- [ ] Fase dan jalur bracket
- [ ] Ranking FIFA
- [ ] Elo rating
- [ ] 5–10 pertandingan terakhir
- [ ] Head-to-head
- [ ] Statistik gol dan kebobolan
- [ ] xG / shot quality jika tersedia
- [ ] Cedera dan suspensi
- [ ] Prediksi line-up
- [ ] Analisis taktik
- [ ] Faktor eksternal
- [ ] Probabilitas
- [ ] Prediksi skor
- [ ] Risiko prediksi
- [ ] Confidence level
