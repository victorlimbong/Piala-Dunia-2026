# Template Input Post-Match Learning

Gunakan template ini setelah pertandingan selesai untuk mengevaluasi prediksi.

```markdown
Evaluasi post-match:

- Kompetisi:
- Tahun:
- Fase:
- Tim A:
- Tim B:
- Prediksi awal:
- Prediksi alternatif:
- Probabilitas awal:
- Hasil aktual:
- Skor akhir:
- Pemenang:
- Gol:
- Kartu merah:
- Momen penentu:
- Statistik utama:
- Catatan tambahan:
```

---

## Contoh Pengisian

```markdown
Evaluasi post-match:

- Kompetisi: Piala Dunia
- Tahun: 2026
- Fase: Round of 32
- Tim A: Brasil
- Tim B: Jepang
- Prediksi awal: Brasil menang 2-1
- Prediksi alternatif: Brasil 2-0, Brasil 1-0, 1-1 extra time
- Probabilitas awal: Brasil 60%, Seri 22%, Jepang 18%
- Hasil aktual: Brasil menang
- Skor akhir: Brasil 2-1 Jepang
- Pemenang: Brasil
- Gol: data gol belum diisi
- Kartu merah: data belum diisi
- Momen penentu: data belum diisi
- Statistik utama: data belum diisi
- Catatan tambahan: evaluasi akurasi, calibration review, error attribution, dan learning note
```

---

## Checklist Evaluasi

- [ ] Apakah pemenang benar?
- [ ] Apakah skor utama tepat?
- [ ] Apakah skor alternatif relevan?
- [ ] Apakah margin gol benar?
- [ ] Apakah over/under benar?
- [ ] Apakah BTTS benar?
- [ ] Apakah clean sheet benar?
- [ ] Apakah tim lolos benar?
- [ ] Apakah probabilitas terlalu tinggi/rendah?
- [ ] Apakah confidence level realistis?
- [ ] Faktor apa yang paling menentukan?
- [ ] Apa yang perlu diperbaiki untuk prediksi berikutnya?
